# AI

This folder contains the AI service integration for NyayaAI.

## Technologies
- OpenRouter API for AI model access
- Mistral AI model for legal assistance

## Files
- `ai.ts` - AI service functions for legal queries

## Configuration
Set the `VITE_OPENROUTER_API_KEY` environment variable in the frontend `.env` file.

## Usage
The AI service provides legal advice in both English and Hindi for Indian legal matters.
